車道線偵測（OpenCV + NumPy）

以經典影像處理管線（ROI → Canny → HoughLinesP）實作即時/離線的車道線偵測與視覺化，並加入簡單的時序穩定化策略，作為 ADAS / 自駕視覺入門基準。

動機（Why）

真實道路上的車道線常因光照、陰影、反光、磨損而不連續，導致偵測結果抖動或短暫丟失。這個專案希望用簡潔、可讀、可調的 OpenCV 管線，建立一個穩定的 baseline：

只專注**道路區域（ROI）**降低雜訊

用 Canny 抓邊緣、HoughLinesP 找線段

透過「上一幀記憶」策略平滑輸出，避免一閃一滅

便於之後對照更進階的方法（Kalman/EMA、學習式語意分割等）

功能特色

ROI 梯形遮罩（可視化檢查）

Canny + HoughLinesP 偵測管線

左/右車道線配對與時序平滑

影片/攝影機即時顯示與儲存

環境需求

Python 3.9+（3.10 可）

OpenCV, NumPy
